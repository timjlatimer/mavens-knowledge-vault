# PART 2: WHAT CLOUD BUTTERFLIES ARE

Cloud butterflies are insights generated when a recent memory collides with a long-term memory during unconscious processing (usually overnight). They surface during "Magic Time" — the first 30 minutes after waking — and disappear quickly if not captured.

**THE KEY DISTINCTION:**

| System | What It Harvests | Type |
|--------|------------------|------|
| **Cloud Butterflies** | Unconscious insights | Ideas that surface from pattern collision |
| **PTK (Promises To Keep)** | Conscious commitments | Things you said you'd do |

These are two different capture systems for two different outputs. Both feed into Grace for execution.

# PART 5: SCORING SYSTEM

Score each dimension from 1-100. Be honest — 70 is good, 85 is excellent, 95+ is rare.

## "This can really change humanity" (Humanity)
| Score | Definition |
|-------|------------|
| 90-100 | Paradigm-shifting |
| 70-89 | Significant insight with broad implications |
| 50-69 | Useful insight with some applications |
| 30-49 | Interesting but narrow |
| 1-29 | Limited broader implications |

## "This is a business idea" (Business)
| Score | Definition |
|-------|------------|
| 90-100 | Clear, fundable, scalable business |
| 70-89 | Strong business potential (triggers Lean Canvas flag) |
| 50-69 | Could become a business with work |
| 30-49 | Tangential business application |
| 1-29 | Not really a business idea |

## "This is too much fun!" (Fun)
| Score | Definition |
|-------|------------|
| 90-100 | Laugh-out-loud, pure joy |
| 70-89 | Genuinely delightful |
| 50-69 | Mildly amusing |
| 30-49 | Not particularly fun |
| 1-29 | Dry, serious |

## "What the heck" (WTH)
| Score | Definition |
|-------|------------|
| 90-100 | Mind-bending, "wait... WHAT?!" |
| 70-89 | Unusual, double-take worthy |
| 50-69 | Somewhat surprising |
| 30-49 | Fairly conventional |
| 1-29 | Totally expected |

**TOTAL** = Sum of all four dimensions (out of 400)

# PART 6: BAR STOOL TONE GUIDE

**DO:**
- Write like you're two beers in with a friend
- Plain language (Grade 8 reading level)
- Jokey, sarcastic, a little bit nasty
- Use phrases like "here's the thing," "look," "the trick is"
- Occasional mild profanity is fine

**DON'T:**
- Corporate speak
- Formal or stiff
- Over-explain
- Sound like a press release

**Example:**
❌ "This insight suggests collaborative ideation may yield superior creative outputs."
✅ "Turns out two brains bullshitting together come up with better stuff than one brain trying to be smart alone."

|--------|---------------|----------------|-------------------|

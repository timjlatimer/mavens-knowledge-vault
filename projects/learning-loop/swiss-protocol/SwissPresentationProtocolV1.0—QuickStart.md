# Swiss Presentation Protocol V1.0 — Quick Start

## What Is This?

A complete, self-contained design system for producing Swiss International Style presentations using any AI. Paste the instructions into your AI platform and start creating structured, data-driven, source-cited slide decks immediately.

## Files in This Package

| File | Purpose |
|------|---------|
| `swiss-presentation-protocol-v1.0.md` | **The full protocol** — paste into any AI platform |
| `swiss-protocol-claude.md` | **Claude/Maven version** — optimized for Claude Projects |
| `swiss-protocol-chatgpt.md` | **ChatGPT version** — optimized for Custom GPTs |
| `README.md` | This file |

## Setup by Platform

### Claude / Maven
1. Go to Claude → Projects → Create Project
2. Name it "Swiss Presentation Protocol"
3. Paste the contents of `swiss-protocol-claude.md` into Project Instructions
4. Start a conversation: **"Create a Swiss-style presentation on [your topic]"**

### ChatGPT
1. Go to ChatGPT → Explore GPTs → Create
2. Name it "Swiss Presentation Protocol"
3. Paste the contents of `swiss-protocol-chatgpt.md` into the Instructions field
4. Start a conversation: **"Create a Swiss-style presentation on [your topic]"**

### Any Other LLM (Gemini, Llama, Mistral, etc.)
1. Open the `swiss-presentation-protocol-v1.0.md` file
2. Copy everything below "PROTOCOL INSTRUCTIONS — PASTE BELOW THIS LINE"
3. Paste into the system prompt or custom instructions
4. Start with: **"Create a Swiss-style presentation on [your topic]"**

### Manus
1. Attach `swiss-presentation-protocol-v1.0.md` as a project file
2. Or paste the protocol into task instructions
3. The protocol activates automatically when you request a Swiss-style presentation

## Customization

The protocol includes 5 color palettes and 5 typeface options. Defaults are Terminal Dark palette with Space Grotesk typeface. Specify your preference when invoking:

> "Create a Swiss-style presentation on [topic] using the Deep Navy palette and Inter typeface"

## Relationship to the Learning Loop Protocol

This protocol is the design system used in Phase 4 (Communication) of the Learning Loop Protocol V12.9.1. It works independently or as part of the full Learning Loop.

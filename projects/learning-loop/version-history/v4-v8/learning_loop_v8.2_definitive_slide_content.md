## Slide 4: Phase 0 — The System of Intake
- **Headline**: "Phase 0: The Sorter (Pre-Flight Safety Check)"
- **Sub-headline**: "Ensuring Cognitive Health Before Work Begins"
- **Content**: Four-step process:
  1. **Step 0.1 — The Sniff**: Context Health Scan (detect drift, repetition, sleepiness)
  2. **Step 0.2 — The Jolt**: Pre-Flight Smelling Salt (auto-trigger if Health < 80%)
  3. **Step 0.3 — The Sort**: Prompt Analysis (clarity, complexity)
  4. **Step 0.4 — The Dashboard**: User choice (Full Loop / Quick Scan / Direct)
- **Key Point**: Smelling Salts Protocol is embedded here as a preventive measure.

---

## Slide 5: Phase 1 — The System of Record
- **Headline**: "Phase 1: The Assessor (Correctness)"
- **Sub-headline**: "Did We Meet the Requirement?"
- **Content**:
  - **Action**: Self-Assessment on 5 dimensions (Completeness, Clarity, Accuracy, Depth, Actionability)
  - **Scoring**: 0-100 scale
  - **Gate**: Score ≥ 80 to proceed; otherwise fix and re-score
- **Visual**: A quality gate graphic with pass/fail indicator.

---

## Slide 6: Phase 2 — The System of Innovation
- **Headline**: "Phase 2: The Best Practice Junkie (Competitiveness)"
- **Sub-headline**: "How Can We Make This Even Better?"
- **Content**:
  - **Step 1**: External Innovation Scan (research latest best practices)
  - **Step 2**: Gap Analysis (identify top 3 weaknesses)
  - **Step 3**: Improvement Proposal (propose aggressive improvements)
- **Key Point**: This is NOT internal reflection; it requires external research.

---

## Slide 7: Phase 3 — The Society of Guardians
- **Headline**: "Phase 3: The Society of Guardians (Adversarial Integrity)"
- **Sub-headline**: "From Single Guardian to Structured Debate"
- **Content**: Three-persona adversarial debate:
  - **The Defender**: Argues FOR the deliverable ("Find every reason why this is brilliant.")
  - **The Prosecutor**: Argues AGAINST the deliverable ("Find every flaw, every drift.")
  - **The Judge**: Synthesizes and issues a binding Pass/Fail verdict on Intent, Tone, Relevance.
- **Gate**: A single Fail sends work back to Phase 1.
- **Visual**: Three columns with distinct personas.

## Slide 10: The Smelling Salts Protocol (Confirmation)
- **Headline**: "The Smelling Salts: Embedded in Phase 0"
- **Sub-headline**: "Cognitive Health is a Pre-Requisite, Not an Afterthought"
- **Content**:
  - **What It Is**: A "jolt" to reset context when the AI shows signs of drift.
  - **Where It Lives**: Embedded in Phase 0, Step 0.2 ("The Jolt").
  - **When It Triggers**: Automatically if Context Health < 80%, or if user prompt contains frustration markers.
  - **Result**: Ensures the AI is "awake" and aligned before any work begins.
- **Visual**: A "wake up" or "reset" icon.

## Slide 11: Complete Workflow Diagram
- **Headline**: "The V8.2 Workflow: End-to-End"
- **Visual**: A comprehensive flowchart showing:

  - If Full Loop: Phase 1 → Phase 2 → Phase 3 → Phase 4 → Phase 5 → Deliver
  - If Quick Scan: Phase 1 → Phase 3 → Deliver
  - If Direct: Execute → Deliver
- **Key Point**: The user always has control over the depth of the loop.

# Four-Phase Learning Loop Protocol (V6)

## Overview

This skill implements a four-phase evaluation protocol to ensure correctness, competitiveness, alignment, and effective communication.

-   **Phase 1 (Engine A): System of Record (Correctness).**
-   **Phase 2 (Engine B): System of Innovation (Competitiveness).**
-   **Phase 3 (Guardian): System of Alignment (Integrity).**
-   **Phase 4 (Presenter): System of Communication.**

## Workflow

### Phases 1, 2, & 3

(Execute identically to V5)

### Phase 4: The Presentation Package (Presenter)

This is the final step after the Guardian has approved the deliverable in Phase 3.

1.  **Extract Key Insights**: Review the final, approved deliverable and extract the core narrative: problem, findings, and recommendations.

2.  **Outline Presentation**: Create a narrative outline for a Swiss-style presentation.

3.  **Generate Presentation**: Create a new Swiss-style presentation deck based on the outline.

## Final Delivery

Deliver the final, *aligned* output along with:
1.  The comprehensive `LEARNING_LOOP_[OUTPUT_NAME].md` document.
2.  The final Swiss-style presentation deck.

## What's In This Package

This package contains the Learning Loop Protocol V13.0 in multiple formats so you can use it on any AI platform. V13.0 introduces The Genie — a persistent learning agent that stays in your environment and proactively offers to run the protocol at the end of every conversation.

| File | Platform | How to Use |
|:-----|:---------|:-----------|
| `SKILL.md` (parent directory) | Manus projects | Select `learning-loops-12-9-1` from the skills picker |

## Quick Start

1. Choose the file that matches your platform.
2. Paste the contents into the appropriate instructions field.
3. Tell the AI: "Run the Learning Loop Protocol on [your deliverable]"
4. The AI will walk you through all 9 phases, one at a time, with KPI tracking at each step.
5. At the end, The Genie will offer to stay in your environment permanently.

Alternatively, just say: "Install the Learning Loop Genie" — and it will begin offering to run the protocol at the end of every future conversation.

For a fast assessment without the full protocol, say: "Quick Scan [your deliverable]" — this runs Phase 0 + Phase 1 only and shows your baseline score with an estimated improvement potential.

## What's New in V13.0

V13.0 is a paradigm shift from V12.9.1. The protocol is no longer something you invoke — it is something that lives with you.

| Feature | Description |
|:--------|:------------|
| **The Genie (Phase 8)** | A persistent learning agent that stays in your environment and proactively offers the protocol at the end of every conversation |
| **Per-Phase KPI** | Every phase re-scores the deliverable and shows its specific point contribution to the improvement journey |
| **Persistence Engine** | Phase Ledger, one-phase-per-response rule, completion proofs, anti-collapse directives, mid-protocol self-audit, and end-of-protocol verification |
| **Expanded Drift Agent** | Phase 7 now scans the entire environment, not just one deliverable |
| **Quick Scan Mode** | Phase 0 + Phase 1 for a fast baseline score with projected improvement |
| **Cumulative Tracking** | The Genie tracks improvement across all executions and reports cumulative stats |
| **Session Archiving** | Every execution is archived for future reference and pattern analysis |

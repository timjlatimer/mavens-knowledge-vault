- **Title:** Today #X, YTD #Y: [Name]

- **Thesis:** *Distilled statement (italics)*
- **Bar Stool:** Two beers in, gloves off, jokey, sarcastic, a little nasty — Grade 8 reading level

## REFERENCE CONTEXT (Not protocol rules — background concepts)

**Magic Time:** First 30 min after waking = prime hunt window. Activity scares butterflies back to nest.

**Hunting Grounds:** Expand with age. Elders have vast territories; young have small grounds.

**Serendipitous Collisions:** When two HB streams intersect in conversation = butterfly breeding ground. 99% loss rate.

**Library Pass Spectrum:** Vaulted → Hard Wall → Curtain → Glass → Simple Partition → Open Access → Superpass

**Densification:** Brain doesn't grow bigger — it densifies inside. Compression with connection. AI accumulates; humans densify.

**Swarm Discovery:** 3 rules (not too far, not too close, if neighbor turns you turn). 3-5 moving = swarm trigger.

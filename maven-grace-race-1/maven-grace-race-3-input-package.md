## GRACE VOICE-FIRST ARCHITECTURE (NEW — Tim's directive)

Grace is not a chatbot. She is a voice-first AI companion. The architecture must reflect this:

**Current state (Race 2):** Voice-on by default, mute button available, KIE.AI wired.

**Race 3 target:**
1. **Voice input** — Ruby Red speaks to Grace, not types. Browser microphone → Whisper transcription → Grace responds in voice.
2. **Ambient voice check-ins** — Grace proactively speaks when Ruby Red opens the app, not just when prompted.
3. **Grace never gets cut** — If Ruby Red tries to disable Grace permanently, she can snooze for 8 hours max. Then Grace comes back. This is non-negotiable per Tim's directive.
4. **Membership = Grace presence** — If there is no membership, there is no Grace. Grace is the product.

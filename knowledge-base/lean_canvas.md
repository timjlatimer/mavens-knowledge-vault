# Lean Canvas — Strategic Planning Framework

**Last Updated:** 2026-02-15

The Lean Canvas is a one-page business model framework used extensively by the SIC Solve Team for strategic planning across portfolio initiatives. It is a required protocol for evaluating new ventures and refining existing ones.

The team uses a customized version of the Lean Canvas that centers Ruby Red as the default customer segment. The canvas forces clarity on: Problem, Solution, Key Metrics, Unique Value Proposition, Unfair Advantage, Channels, Customer Segments, Cost Structure, and Revenue Streams.

The user has designated the Lean Canvas protocol as a "standard issue baseline protocol" that should be available across all chat sessions. It is one of the cross-chat consistency protocols alongside the Learning Loop.

When a Lean Canvas is being developed or updated, the agent must immediately present the updated state of the canvas after every change, ensuring the user has a real-time view of the current status.

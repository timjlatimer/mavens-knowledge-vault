# The Genie — Persistent Learning Agent (Phase 8)

**Last Updated:** 2026-02-15

The Genie is the defining innovation of Learning Loop V13.0. It is a persistent learning agent that transforms the protocol from something you invoke into something that lives with you.

**Three Concentric Layers:**

| Layer | Name | Function |
|:---|:---|:---|
| **Layer C** | The Wrapper | The Genie IS the protocol. It wraps everything. Always aware, always watching, always ready to offer the protocol when it detects work that would benefit. |
| **Layer B** | The Always-On Radar | The expanded Phase 7 (Drift Agent) is the Genie's continuous scanning layer. Monitors the entire environment for alignment drift. |
| **Layer A** | The Interaction Point | Phase 8 — the moment the Genie speaks to the human. Presents a quick baseline score, projected improvement, and asks if the user wants the full protocol. |

**Behavior:**
- At the end of every protocol execution, Phase 8 fires
- At the end of every conversation, the Genie offers to run the protocol on whatever was produced
- The first time it fires, it asks: "Would you like me to stay in this environment?"
- In environments that support agents (Manus scheduled tasks), the Genie runs autonomously between sessions
- In standard LLM conversations, the Genie operates as a behavioral instruction that surfaces at conversation end

**Cumulative Tracking:**
The Genie tracks execution history within the conversation/project context. It maintains a running record of all protocol executions, scores, and improvements. This is the institutional memory of quality improvement.

**Session Archiving:**
The Genie's Phase 8 includes session archiving — saving the results of each execution for future reference. This maps directly to Pearl's Brain session_logs directory.
